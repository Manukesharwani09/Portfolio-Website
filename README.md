# Manu Kesharwani — Portfolio

Personal portfolio site built with React, TypeScript, and Vite. Terminal-themed aesthetic with a CRT overlay, green-on-black palette, and devlog-style experience writeups.

## Sections

- **Hero** — animated typewriter intro
- **About** — background and stack overview
- **Experience** — internship cards with links to detailed devlogs
- **Projects** — selected builds with tech badges and links
- **Skills** — categorised tech stack
- **LeetCode** — competitive programming stats
- **Books** — reading list
- **Certificates** — certifications
- **Contact** — links and contact form

## Experience / Devlogs

Each internship has a dedicated devlog page (`/experience/:id`) with a problem → solution breakdown for every feature shipped.

| Company | Role | Period |
|---|---|---|
| Tap Invest | Backend Developer Intern | Jun 2026 – Present |
| Melento (formerly Signdesk) | Software Engineer Intern | Jan 2026 – Present |

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Bundler:** Vite
- **Routing:** React Router v6
- **3D / Animation:** Three.js via `@react-three/fiber` + `@react-three/drei`
- **Icons:** Lucide React
- **Styling:** Tailwind CSS

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
